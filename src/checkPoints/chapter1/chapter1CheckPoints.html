Chapter 1 Checkpoint Questions and Answers

Answer: The CPU understands machine language, which is composed of binary code (0s and 1s).

Answer: Assembly language is a low-level programming language that is closely related to machine language but is more readable by humans. It uses symbolic code and is specific to a computer's architecture.

Answer: An assembler is a program that translates assembly language code into machine code so that it can be executed by the CPU.

Answer: A high-level programming language is a language that is more abstract and closer to human languages, making it easier to read and write. It is independent of the hardware and is translated into machine code by a compiler or interpreter.

Answer: A source program is the original code written by a programmer in a high-level programming language before it is compiled or interpreted.

Answer: An interpreter is a program that directly executes the instructions in a source program without compiling it into machine code first, interpreting the code line by line.

Answer: A compiler is a program that translates code written in a high-level programming language into machine code, producing an executable file that can be run by the CPU.

Answer: The difference lies in how the code is executed. In an interpreted language, the code is executed line by line by an interpreter. In a compiled language, the entire program is first compiled into machine code, and then the resulting executable file is run.

Answer: An operating system (OS) is system software that manages computer hardware and software resources, and provides services for computer programs. Popular operating systems include Windows, macOS, Linux, Android, and iOS.

Answer: The major responsibilities include process management, memory management, file system management, device management, security and access control, and providing a user interface.

Answer: Multiprogramming is the ability of an OS to run multiple programs simultaneously by keeping several in memory. Multithreading is the ability of a CPU to execute multiple threads concurrently within a single process. Multiprocessing is the use of two or more CPUs within a computer system to run multiple processes simultaneously.

Answer: Java was invented by James Gosling in 1995 while he was working at Sun Microsystems. Java is now owned by Oracle Corporation.

Answer: A Java applet is a small Java program that can be embedded in a web page and run in a web browser. However, due to security issues, applets are now largely obsolete.

Answer: Android primarily uses Java, and more recently, Kotlin is also officially supported for Android development.

Answer: The Java language specification is the official document that defines the syntax and semantics of the Java programming language.

Answer: JDK stands for Java Development Kit.

Answer: IDE stands for Integrated Development Environment.

Answer: NetBeans and Eclipse are not languages; they are Integrated Development Environments (IDEs) that support Java development. They provide tools and features to write, debug, and manage Java code.

Answer: A keyword is a reserved word in a programming language that has a specific meaning and cannot be used as an identifier. Examples of Java keywords include class, public, static, void, if, else, and return.

Answer: Yes, Java is case sensitive. Java keywords must be written in lowercase.

Answer: A comment is a note in the source code that explains the code and is ignored by the compiler. A single-line comment is denoted by //, and a multi-line comment (comment paragraph) is denoted by /* and */.

Answer: The statement to display a string on the console is System.out.println("your string here");

Answer: The Java source filename extension is .java, and the Java bytecode filename extension is .class.

Answer: The input of a Java compiler is a .java source file, and the output is a .class bytecode file.

Answer: The command to compile a Java program is javac ProgramName.java

Answer: The command to run a Java program is java ProgramName

Answer: JVM stands for Java Virtual Machine, which is a virtual machine that runs Java bytecode on any platform.

Answer: Yes, Java can run on any machine that has a JVM installed. To run Java on a computer, the Java Runtime Environment (JRE) is needed.

Answer: A NoClassDefFoundError occurs when the JVM cannot find the .class file that it needs to run the program. This could be due to an incorrect CLASSPATH or the .class file being missing.

Answer: A NoSuchMethodError occurs when the JVM cannot find the method that it is trying to invoke. This often happens if the main method is incorrectly defined or missing.

Answer: Syntax errors are errors in the code that violate the rules of the programming language and prevent the code from compiling. Runtime errors are errors that occur while the program is running, causing it to crash. Logic errors are errors in the program's logic that cause it to produce incorrect results, even though it compiles and runs.

Answer:
- Syntax Error: Missing semicolon in a statement like System.out.println("Hello World")
- Runtime Error: Dividing by zero in a statement like int result = 10 / 0;
- Logic Error: Incorrectly calculating the sum by using subtraction: int sum = a - b;
